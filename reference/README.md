# Referências visuais

O HTML desta pasta é uma referência visual não normativa para a emissão física. Ele demonstra identidade, proporções e composição, mas contém dados de demonstração e não substitui schemas, ADR-0010 ou `specs/PHYSICAL_DASHBOARD_TWO_FACES.md`.

A implementação final deve ser dinâmica, segura, baseada em PrintSnapshot e sem dados hardcoded.
