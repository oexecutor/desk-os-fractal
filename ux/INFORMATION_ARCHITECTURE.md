# Arquitetura da informação

## Navegação primária

- Intake
- Projetos
- Foco
- Sprint
- Emitir
- Histórico
- Configurações

## Hierarquia de trabalho

```text
Workspace
├── Portfólio (opcional)
│   └── Projetos
└── Projeto único
    └── Fases/Workflows/Semanas/Blocos
        └── Ações + LINK
```

## Regra de contexto

Toda tela operacional exibe:

- workspace;
- projeto;
- versão do plano;
- breadcrumb;
- estado de sincronização;
- lifecycle do plano quando não ativo.
