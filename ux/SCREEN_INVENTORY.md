# Inventário de telas

| Tela | Objetivo | Ação dominante |
|---|---|---|
| Intake | Receber projeto | Adicionar conteúdo |
| Extraction Review | Conferir o que foi lido | Confirmar contexto |
| Decomposition Progress | Mostrar etapas reais | Aguardar/cancelar |
| Plan Review | Revisar estrutura e lacunas | Aprovar ou corrigir |
| Portfolio Root | Selecionar projeto | Abrir projeto |
| Project Root | Navegar na escala macro | Abrir nó |
| Node Zoom | Navegar filhos | Abrir próximo nível |
| Focus | Executar | Concluir próxima ação |
| Sprint | Ver semana | Selecionar bloco/dia |
| Emit Preview | Conferir duas faces | Gerar snapshot |
| QR Resolve | Entender QR | Abrir ou confirmar |
| Close Day | Encerrar bloco | Registrar fechamento |
| Recycle | Decidir continuidade | Selecionar decisão |
| History | Auditar eventos/versões | Abrir detalhe |
| Settings | Preferências e sync | Salvar configuração |
