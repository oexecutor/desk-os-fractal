# Definition of Done

Uma tarefa só está concluída quando:

- requisito e critério de aceite estão identificados;
- código possui tipos estritos;
- schemas e fixtures foram atualizados quando aplicável;
- testes relevantes passam;
- acessibilidade foi verificada em mudanças de UI;
- nenhum segredo foi introduzido;
- erros são estruturados;
- documentação pública do módulo foi atualizada;
- decisão arquitetural nova possui ADR;
- build e lockfile são reproduzíveis;
- não há `TODO` crítico oculto;
- evidências do gate foram registradas.
