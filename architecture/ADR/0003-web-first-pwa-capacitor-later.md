# ADR-0003 — Web-first, PWA primeiro e Capacitor depois

**Status:** Aceito

## Decisão

Consolidar domínio, frontend e sincronização na web. Capacitor será uma casca de distribuição após os testes end-to-end web estarem estáveis.

## Consequências

Evita duas implementações divergentes e preserva possibilidade de Android/iOS.
