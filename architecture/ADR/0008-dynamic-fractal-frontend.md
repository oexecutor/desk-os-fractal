# ADR-0008 — Frontend fractal com cardinalidade dinâmica

**Status:** Aceito

## Decisão

Projeto único e portfólio usam o mesmo navegador recursivo. Grades de 3, 5 ou 9 são layouts, não domínio. O nível operacional padrão mantém 3 ações + LINK.

## Consequências

Componentes precisam suportar listas variáveis, paginação/condensação e foco acessível.
